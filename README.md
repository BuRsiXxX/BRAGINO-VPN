## ✨ Основные возможности

| Функция | Описание |
|---------|----------|
| 🚀 **Быстрое подключение** | Мгновенное подключение к VPN с минимальными задержками |
| 🌍 **Глобальный режим** | Весь трафик ПК направляется через VPN-туннель |
| 🛡️ **Режим туннелирования** | Гибкая настройка — только выбранные приложения или домены через VPN |
| 🚫 **Режим исключений** | Указанные приложения и домены идут напрямую, остальное — через VPN |
| 🇷🇺 **Обход для .RU** | Российские сайты автоматически обходят VPN для максимальной скорости |
| 🔗 **Подписки** | Поддержка VLESS-ключей и подписок с автообновлением |
| 🌐 **Кастомный DNS** | Встроенные пресеты Cloudflare, Google и AdGuard DNS |
| 📊 **Мониторинг пинга** | Отображение задержки до каждого сервера в реальном времени |
| 🔄 **Автоподключение** | Автоматическое подключение к VPN при запуске клиента |
| 🖥️ **Автозапуск** | Запуск клиента вместе с Windows |
| 📝 **Логирование** | Сохранение логов для диагностики |
| 🎨 **Современный дизайн** | Стильный тёмный интерфейс с анимациями |

## 📥 Установка

### Системные требования

- **ОС:** Windows 10 / 11 (64-bit)
- **Права:** Запуск от имени **Администратора** (необходимо для TUN-адаптера)

### Как скачать и установить

1. Перейдите в раздел **[Releases](../../releases)** этого репозитория
2. Скачайте последнюю версию:
   - `bragino-vpn_0.4.0_x64-setup.exe` — установщик EXE  
   - `bragino-vpn_0.4.0_x64_en-US.msi` — установщик MSI
3. Запустите скачанный файл
4. Следуйте инструкциям установщика
5. **Важно:** Запускайте приложение **от имени Администратора**

---

## 🚀 Быстрый старт

1. 🔧 Откройте **BRAGINO VPN**
2. ➕ Нажмите `+` рядом с выбором сервера и вставьте **VLESS-ключ** или **ссылку на подписку**
3. 📡 Выберите сервер из списка
4. ⚡ Нажмите кнопку **подключения** — готово!

## 🔐 Безопасность

- Используется протокол **VLESS + Reality** — один из самых защищённых VPN-протоколов
- Трафик шифруется и маскируется под обычный HTTPS
- Поддержка TUN-интерфейса обеспечивает туннелирование на уровне системы
