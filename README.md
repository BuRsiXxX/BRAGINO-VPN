**BRAGINO VPN** — это высокопроизводительный, VPN-клиент для Windows, обеспечивающий ультрабыстрое, безопасное и стабильное подключение через передовой протокол **VLESS + Reality (Wintun TUN-интерфейс)**.

## ✨ Основные возможности (v0.5.0)

| Функция | Описание |
|---------|----------|
| 🚀 **Быстрое подключение** | Мгновенный запуск VPN-соединения с минимальными накладными расходами |
| 🌍 **Глобальный режим** | Полное системное туннелирование (весь трафик ПК идет через VPN) |
| 🛡️ **Режим туннелирования** | Гибкое включение — только выбранные приложения или доменные зоны работают через VPN |
| 🚫 **Режим исключений** | Белый список — указанные программы и сайты идут напрямую, всё остальное — в VPN |
| 🇷🇺 **Автообход для .RU** | Автоматическое исключение российских зон (`.ru`, `.su`, `.рф`) для максимальной скорости |
| 🔗 **Подписки и Ключи** | Поддержка VLESS-ключей и внешних ссылок подписок с автообновлением списков |
| 🌐 **Кастомный DNS** | Интеграция быстрых DNS-провайдеров: Cloudflare, Google и защищенный AdGuard |
| 📊 **Мониторинг пинга** | Измерение и отображение задержки (Latency) до каждого сервера в реальном времени |
| 🔄 **Автоподключение** | Опция автоматического подключения к последнему серверу при запуске приложения |
| 🖥️ **Автозапуск** | Запуск клиента вместе с Windows в фоновом режиме |
| 📝 **Логирование** | Подробный сбор логов Xray и системных событий для диагностики |
| 🎨 **UI дизайн** | Роскошный тёмный интерфейс со сглаженными неоновыми обводками и анимациями |

## 📥 Установка

### Системные требования

- **ОС:** Windows 10 / 11 (64-bit)
- **Права:** Запуск от имени **Администратора** (необходимо для создания сетевого адаптера Wintun и управления таблицей маршрутизации Windows)

### Как скачать и установить

1. Перейдите в раздел **[Releases](../../releases)** этого репозитория.
2. Скачайте последнюю версию:
   * `BRAGINO VPN_0.5.0_x64-setup.exe`
3. Запустите скачанный файл и выполните установку.
4. **Важно:** Всегда запускайте ярлык приложения **от имени Администратора** (для этого в свойствах ярлыка можно установить галочку постоянного запуска от администратора).

## 🚀 Быстрый старт

1. 🔧 Откройте **BRAGINO VPN**.
2. ➕ Нажмите кнопку `+` в блоке выбора сервера и вставьте ваш **VLESS-ключ** или **ссылку на подписку**.
3. 📡 Выберите желаемый сервер из выпадающего списка.
4. ⚡ Нажмите на круглую центральную кнопку подключения — анимация закрутится, и соединение будет установлено!

---

## 🔐 Безопасность

* Протокол **VLESS + Reality** маскирует VPN-трафик под стандартный защищенный веб-серфинг (HTTPS), делая его невидимым для глубокого анализа пакетов (DPI).
* Автоматический сброс маршрутов и очистка TUN-адаптера при отключении гарантируют, что в системе не останется «зависшего» трафика.
